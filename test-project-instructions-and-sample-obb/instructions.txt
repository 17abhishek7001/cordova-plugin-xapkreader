(Using Windows and the C: drive)

1) Make sure a physical phone is connected to your work computer. Emulation will likely not work.

2) (Command line) C:

3) (Command line) cordova create C:/testproject com.testproject TestProject

4) (Command line) cd C:/testproject

5) (Command line) cordova platform add android

6) (Command line) cordova plugin add https://github.com/agamemnus/cordova-plugin-xapkreader.git

7) Create a STORE zip with some file (a funny image, perhaps).

8) Add a zip STORE named main.x.com.testproject.obb into "Phone/Android/obb/com.testproject/". (e.g.: main.1.com.testproject.obb)

9) Modify android.json:
a) Change the xapk_expansion_authority string value from "com.sample.expansion" to "com.testproject.expansion".
b) Modify xapk_google_play_public_key from "YOUR_GOOGLE_PLAY_LICENSE_KEY" to your Google Play public key.

10) Add this to /platforms/android/project.properties:
android.library.reference.1=../../plugins/com.flyingsoftgames.xapkreader/android-sdk/extras/google/play_apk_expansion/downloader_library

11) Access the funny image like this: <img src="content://com.testproject.expansion/hoarder.png">. Change the /index.html file to add this line.

12) (Command line) cordova run
